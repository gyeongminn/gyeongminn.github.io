word2vecмқ„ мқҙмҡ©н•ң лӢЁм–ҙ мһ„лІ л”©

м¶”лЎ  кё°л°ҳ кё°лІ•кіј лӢЁмҲңн•ң word2vecмқҳ кө¬нҳ„

Dec 19, 2023

ліё нҸ¬мҠӨнҢ…мқҖ ‘л°‘л°”лӢҘл¶Җн„° мӢңмһ‘н•ҳлҠ” л”Ҙлҹ¬лӢқ 2’ көҗмһ¬лҘј м°ёкі н–ҲмҠөлӢҲлӢӨ.

м¶”лЎ  кё°л°ҳ кё°лІ•кіј мӢ кІҪл§қ

лӢЁм–ҙлҘј лІЎн„°лЎң н‘ңнҳ„н•ҳлҠ” л°©лІ•мқҖ нҶөкі„ кё°л°ҳ кё°лІ•кіј м¶”лЎ  кё°л°ҳ кё°лІ•мқҙ мһҲлӢӨ. л‘ҳ лӘЁл‘җ л¶„нҸ¬ к°Җм„Өмқ„ кё°л°ҳмңјлЎң н•ңлӢӨ.

нҶөкі„ кё°л°ҳ кё°лІ•мқҳ л¬ём ңм җ

нҶөкі„ кё°л°ҳ кё°лІ•мқҖ мЈјліҖ лӢЁм–ҙмқҳ л№ҲлҸ„лҘј кё°мҙҲлЎң лӢЁм–ҙлҘј н‘ңнҳ„н•ңлӢӨ. лӢЁм–ҙ мҲҳк°Җ $N$к°ңмқј л•Ң, $N \times N$мқҙлқјлҠ” кұ°лҢҖн•ң н–үл ¬мқ„ л§Ңл“ӨкІҢ лҗңлӢӨ. мҳҒм–ҙ м–ҙнңҳл§Ң н•ҙлҸ„ 100л§Ң к°ңм—җ к°Җк№ҢмҡҙлҚ°, к·ёл ҮлӢӨл©ҙ 1мЎ°к°ңмқҳ мӣҗмҶҢлҘј к°Җм§„ н–үл ¬мқҙ н•„мҡ”н•ҳлӢӨлҠ” кІғмқҙлӢӨ.

мқҙл ҮкІҢ нҶөкі„ кё°л°ҳ кё°лІ•мІҳлҹј н•ҷмҠө лҚ°мқҙн„°лҘј н•ңлІҲм—җ мІҳлҰ¬н•ҳм§Җ л§җкі , лҚ°мқҙн„°лҘј мһ‘кІҢ лӮҳлҲ  мҲңм°Ём ҒмңјлЎң н•ҷмҠөмӢңнӮӨлҠ” (лҜёлӢҲл°°м№ҳ н•ҷмҠө) л°©лІ•мқҙ н•„мҡ”н•ҳлӢӨ.

м¶”лЎ  кё°л°ҳ кё°лІ•мқҳ к°ңмҡ”

м¶”лЎ мқҙлһҖ мЈјліҖ лӢЁм–ҙ (л§ҘлқҪ)к°Җ мЈјм–ҙмЎҢмқ„ л•Ң, л¬ҙмҠЁ лӢЁм–ҙк°Җ л“Өм–ҙк°Ҳ м§Җ лӢЁм–ҙлҘј мң м¶”н•ҳлҠ” кІғмқҙлӢӨ.

л§ҘлқҪ м •ліҙлҘј мһ…л Ҙл°ӣм•„ м¶ңнҳ„н•  мҲҳ мһҲлҠ” лӢЁм–ҙл“Өмқҳ нҷ•лҘ л¶„нҸ¬лҘј лӮҳнғҖлӮҙлҠ” лӘЁлҚёмқ„ л§Ңл“Өкі , н•ҷмҠөмқҳ кІ°кіјлЎң л¶„мӮ° н‘ңнҳ„мқ„ м–»лҠ” кІғмқҙ м¶”лЎ  кё°л°ҳ кё°лІ•мқҙлӢӨ.

мӢ кІҪл§қм—җм„ңмқҳ лӢЁм–ҙ мІҳлҰ¬

мӢ кІҪл§қмқҖ лӢЁм–ҙлҘј к·ёлҢҖлЎң мІҳлҰ¬н•  мҲҳ м—Ҷкё° л•Ңл¬ём—җ, лӢЁм–ҙлҘј кі м •лҗң кёёмқҙмқҳ лІЎн„°лЎң ліҖнҷҳн•ҙм•ј н•ңлӢӨ. мқҙлҘј мң„н•ҙ к°ҖмһҘ лҢҖн‘ңм ҒмңјлЎң мӮ¬мҡ©лҗҳлҠ” л°©лІ•мқҙ мӣҗн•« мқёмҪ”л”©(one-hot encoding)мқҙлӢӨ.

лӢЁм–ҙ(н…ҚмҠӨнҠё) лӢЁм–ҙ ID мӣҗн•« н‘ңнҳ„
you 0 (1, 0, 0, 0, 0, 0, 0, 0)
goodbye 2 (0, 0, 1, 0, 0, 0, 0, 0)

мң„мҷҖ к°ҷмқҙ лӢЁм–ҙлҠ” н…ҚмҠӨнҠё, лӢЁм–ҙ ID, мӣҗн•« н‘ңнҳ„мңјлЎң лӮҳнғҖлӮј мҲҳ мһҲлӢӨ. лӢЁм–ҙлҘј кі м • нҒ¬кё°мқҳ мӣҗн•« н‘ңнҳ„мңјлЎң лӮҳнғҖлӮҙкІҢ лҗҳл©ҙ лүҙлҹ°мқҳ мҲҳлҘј кі м •н•  мҲҳ мһҲлӢӨ.

мӢ кІҪл§қмқ„ кө¬м„ұн•ҳлҠ” кі„мёөл“Өмқҙ лІЎн„°лҘј мІҳлҰ¬н•  мҲҳ мһҲмңјлҜҖлЎң, мқҙм ң лӢЁм–ҙлҘј мӢ кІҪл§қмңјлЎң мІҳлҰ¬н•  мҲҳ мһҲмқ„ кІғмқҙлӢӨ.

мҷ„м „м—°кІ°кі„мёөмқҳ кі„мӮ°мқҖ н–үл ¬ кіұмңјлЎң мҲҳн–үн•  мҲҳ мһҲкі . н–үл ¬ кіұмқҖ л„ҳнҢҢмқҙмқҳ np.matmul()лЎң н•  мҲҳ мһҲлӢӨ.

CBOW

CBOW лӘЁлҚёмқҳ м¶”лЎ  мІҳлҰ¬

CBOW лӘЁлҚёмқҖ л§ҘлқҪмңјлЎңл¶Җн„° нғҖк№ғмқ„ м¶”мёЎн•ҳлҠ” мҡ©лҸ„мқҳ мӢ кІҪл§қмқҙлӢӨ. (нғҖк№ғмқҖ мӨ‘м•ҷ лӢЁм–ҙ, л§ҘлқҪмқҖ мЈјліҖ лӢЁм–ҙлҘј мқҳлҜён•ңлӢӨ)

мһ…л Ҙмёөмқҙ 2к°ң мһҲкі , мқҖлӢүмёөмқ„ кұ°міҗ м¶ңл Ҙмёөм—җ лҸ„лӢ¬н•ңлӢӨ.

л‘җ мһ…л Ҙмёөм—җм„ң мқҖлӢүмёөмңјлЎңмқҳ ліҖнҷҳмқҖ мҷ„м „м—°кІ°кі„мёөмқҙ мҲҳн–үн•ңлӢӨ. к·ёлҰ¬кі  мқҖлӢүмёөм—җм„ң м¶ңл Ҙмёө лүҙлҹ°мңјлЎңмқҳ ліҖнҷҳмқҖ лӢӨлҘё мҷ„м „м—°кІ°кі„мёөмқҙ мІҳлҰ¬н•ңлӢӨ. мһ…л Ҙмёөмқҙ м—¬лҹ¬ к°ңмқҙл©ҙ м „мІҙлҘј нҸүк· н•ҳл©ҙ лҗңлӢӨ.

м¶ңл Ҙмёөмқҳ лүҙлҹ°мқҖ мҙқ 7к°ңмқёлҚ°, мқҙ лүҙлҹ° н•ҳлӮҳн•ҳлӮҳк°Җ к°Ғк°Ғмқҳ лӢЁм–ҙм—җ лҢҖмқ‘н•ңлӢӨ. м¶ңл Ҙмёө лүҙлҹ°мқҖ к°Ғ лӢЁм–ҙмқҳ м җмҲҳлҘј лң»н•ҳл©°, к°’мқҙ лҶ’мқ„мҲҳлЎқ лҢҖмқ‘ лӢЁм–ҙмқҳ м¶ңнҳ„ нҷ•лҘ лҸ„ лҶ’м•„м§„лӢӨ. мқҙ м җмҲҳм—җ мҶҢн”„нҠёл§ҘмҠӨ н•ЁмҲҳлҘј м Ғмҡ©н•ҙм„ң, нҷ•лҘ мқ„ м–»мқ„ мҲҳ мһҲлӢӨ.

н•ҷмҠөмқ„ м§„н–үн• мҲҳлЎқ л§ҘлқҪм—җм„ң м¶ңнҳ„н•ҳлҠ” лӢЁм–ҙлҘј мһҳ м¶”мёЎн•ҳлҠ” л°©н–ҘмңјлЎң мқҙ л¶„мӮ° н‘ңнҳ„л“Өмқҙ к°ұмӢ лҗңлӢӨ. мқҙл ҮкІҢ м–»мқҖ лІЎн„°м—җлҠ” лӢЁм–ҙмқҳ мқҳлҜёлҸ„ нҸ¬н•Ёлҗҳм–ҙ мһҲлӢӨ.

class MatMul:
    def __init__(self, W):
        self.params = [W]
        self.grads = [np.zeros_like(W)]
        self.x = None

    def forward(self, x):
        W, = self.params
        out = np.dot(x, W)
        self.x = x
        return out

    def backward(self, dout):
        W, = self.params
        dx = np.dot(dout, W.T)
        dW = np.dot(self.x.T, dout)
        self.grads[0][...] = dW
        return dx

CBOW лӘЁлҚёмқҖ нҷңм„ұнҷ” н•ЁмҲҳлҘј мӮ¬мҡ©н•ҳм§Җ м•ҠлҠ” к°„лӢЁн•ң кө¬м„ұмқҳ мӢ кІҪл§қмқҙлӢӨ.

CBOW лӘЁлҚёмқҳ н•ҷмҠө

лӘЁлҚёмқҙ мҳ¬л°”лҘё мҳҲмёЎмқ„ н•  мҲҳ мһҲлҸ„лЎқ к°ҖмӨ‘м№ҳлҘј мЎ°м •н•ҙм•ј н•ңлӢӨ.

мҶҢн”„нҠёл§ҘмҠӨ н•ЁмҲҳлҘј мқҙмҡ© н•ҙ м җмҲҳлҘј нҷ•лҘ лЎң ліҖнҷҳн•ҳкі , к·ё нҷ•лҘ кіј м •лӢө л Ҳмқҙлё”лЎңл¶Җн„° көҗм°Ё м—”нҠёлЎңн”ј мҳӨм°ЁлҘј кө¬н•ң нӣ„, к·ё к°’мқ„ мҶҗмӢӨлЎң мӮ¬мҡ©н•ҙ н•ҷмҠөмқ„ м§„н–үн•ңлӢӨ.

м•һм„ң кө¬нҳ„н•ң м¶”лЎ  мІҳлҰ¬лҘј мҲҳн–үн•ҳлҠ” CBOW лӘЁлҚём—җ Softmax кі„мёөкіј Cross Entropy Error кі„мёөмқ„ м¶”к°Җн•ҳкё°л§Ң н•ҳл©ҙ лҗңлӢӨ.

word2vecмқҳ к°ҖмӨ‘м№ҳмҷҖ л¶„мӮ° н‘ңнҳ„

word2vecм—җм„ң мӮ¬мҡ©лҗҳлҠ” мӢ кІҪл§қм—җлҠ” л‘җ к°Җм§Җ к°ҖмӨ‘м№ҳк°Җ мЎҙмһ¬н•ңлӢӨ.

  1. мһ…л Ҙ мёЎ к°ҖмӨ‘м№ҳ $ W_\text{in} $: мһ…л Ҙ мёЎ мҷ„м „м—°кІ°кі„мёөмқҳ к°ҖмӨ‘м№ҳлЎң, к°Ғ н–үмқҖ н•ҙлӢ№ лӢЁм–ҙмқҳ л¶„мӮ° н‘ңнҳ„мқ„ лӮҳнғҖлӮёлӢӨ.

  2. м¶ңл Ҙ мёЎ к°ҖмӨ‘м№ҳ $ W_\text{out} $: м¶ңл Ҙ мёЎ мҷ„м „м—°кІ°кі„мёөмқҳ к°ҖмӨ‘м№ҳлЎң, лӢЁм–ҙмқҳ мқҳлҜёк°Җ мқёмҪ”л”©лҗң лІЎн„°к°Җ к°Ғ м—ҙм—җ м ҖмһҘлҗңлӢӨ.

мқјл°ҳм ҒмңјлЎң word2vecм—җм„ңлҠ” мһ…л Ҙ мёЎ к°ҖмӨ‘м№ҳ $ W_\text{in} $л§Ңмқ„ мөңмў… лӢЁм–ҙмқҳ л¶„мӮ° н‘ңнҳ„мңјлЎң мӮ¬мҡ©н•ңлӢӨ. м¶ңл Ҙ мёЎ к°ҖмӨ‘м№ҳлҠ” лҢҖл¶Җл¶„мқҳ м—°кө¬м—җм„ң лІ„л Өм§„лӢӨ.

л‘җ к°ҖмӨ‘м№ҳлҠ” н•ҳлӮҳл§Ң мқҙмҡ©н•  мҲҳлҸ„ мһҲкі , л‘ҳмқ„ н•©міҗм„ң мӮ¬мҡ©н•  мҲҳлҸ„ мһҲлӢӨ.

word2vecмқҳ skip-gram л“ұ л§ҺмқҖ м—°кө¬м—җм„ңлҠ” мһ…л Ҙ мёЎмқҳ к°ҖмӨ‘м№ҳл§Ң мӮ¬мҡ©н•ҳкі , GloVeм—җм„ңлҠ” л‘җ к°ҖмӨ‘м№ҳлҘј лҚ”н•ҳм—¬ мӮ¬мҡ©н•ңлӢӨ.

н•ҷмҠө лҚ°мқҙн„° мӨҖл№„

“You say goodbye and I say hello” л¬ёмһҘмқ„ мқҙмҡ©н•ҙ н•ҷмҠөмқ„ м§„н–үн•ҙ ліҙмһҗ.

л¬ёмһҘмқ„ м „мІҳлҰ¬н•ҳлҠ” preprocess н•ЁмҲҳлҠ” м—¬кё°лҘј м°ёкі н•ҳмһҗ.

л§ҘлқҪкіј нғҖк№ғ

word2vecм—җм„ң мқҙмҡ©н•ҳлҠ” мӢ кІҪл§қмқҳ мһ…л ҘмқҖ л§ҘлқҪмқҙлӢӨ. к·ё м •лӢө л Ҳмқҙлё”мқҖ мӨ‘м•ҷ лӢЁм–ҙмқё нғҖк№ғ мқҙлӢӨ. мҡ°лҰ¬лҠ” л§ҘлқҪмқ„ мһ…л Ҙн–Ҳмқ„ л•Ң, нғҖк№ғмқ„ м¶ңл Ҙн•  нҷ•лҘ мқҙ лҶ’м•„м§ҖлҸ„лЎқ н•ҷмҠөмӢңнӮӨл©ҙ лҗңлӢӨ.

def create_contexts_target(corpus, window_size=1):
    target = corpus[window_size:-window_size]
    contexts = []

    for idx in range(window_size, len(corpus)-window_size):
        cs = []
        for t in range(-window_size, window_size + 1):
            if t == 0:
                continue
            cs.append(corpus[idx + t])
        contexts.append(cs)

    return np.array(contexts), np.array(target)

    N = corpus.shape[0]

    if corpus.ndim == 1:
        one_hot = np.zeros((N, vocab_size), dtype=np.int32)
        for idx, word_id in enumerate(corpus):
            one_hot[idx, word_id] = 1

        one_hot = np.zeros((N, C, vocab_size), dtype=np.int32)
        for idx_0, word_ids in enumerate(corpus):
            for idx_1, word_id in enumerate(word_ids):
                one_hot[idx_0, idx_1, word_id] = 1

    return one_hot

class SimpleCBOW:
    def __init__(self, vocab_size, hidden_size):
        V, H = vocab_size, hidden_size

        # к°ҖмӨ‘м№ҳ мҙҲкё°нҷ”
        W_in = 0.01 * np.random.randn(V, H).astype('f')
        W_out = 0.01 * np.random.randn(H, V).astype('f')

        # кі„мёө мғқм„ұ
        self.in_layer0 = MatMul(W_in)
        self.in_layer1 = MatMul(W_in)
        self.out_layer = MatMul(W_out)
        self.loss_layer = SoftmaxWithLoss()

        # лӘЁл“  к°ҖмӨ‘м№ҳмҷҖ кё°мҡёкё°лҘј лҰ¬мҠӨнҠём—җ лӘЁмқҖлӢӨ.
        layers = [self.in_layer0, self.in_layer1, self.out_layer]
        self.params, self.grads = [], []
        for layer in layers:
            self.params += layer.params
            self.grads += layer.grads

        # мқёмҠӨн„ҙмҠӨ ліҖмҲҳм—җ лӢЁм–ҙмқҳ л¶„мӮ° н‘ңнҳ„мқ„ м ҖмһҘн•ңлӢӨ.
        self.word_vecs = W_in

    def forward(self, contexts, target):
        h0 = self.in_layer0.forward(contexts[:, 0])
        h1 = self.in_layer1.forward(contexts[:, 1])
        h = (h0 + h1) * 0.5
        score = self.out_layer.forward(h)
        loss = self.loss_layer.forward(score, target)
        return loss

    def backward(self, dout=1):
        ds = self.loss_layer.backward(dout)
        da = self.out_layer.backward(ds)
        da *= 0.5
        self.in_layer1.backward(da)
        self.in_layer0.backward(da)
        return None

word2vec ліҙм¶©

CBOW лӘЁлҚёкіј нҷ•лҘ 

мӮ¬кұҙ Aк°Җ мқјм–ҙлӮ  нҷ•лҘ мқҖ $P(A)$мҷҖ к°ҷмқҙ н‘ңкё°н•ҳкі , AмҷҖ Bк°Җ лҸҷмӢңм—җ мқјм–ҙлӮ  нҷ•лҘ мқҖ $P(A, B)$мҷҖ к°ҷмқҙ н‘ңкё°н•ңлӢӨ. мӮ¬кұҙ Bк°Җ мқјм–ҙлӮ¬мқ„ л•Ң мӮ¬кұҙ Aк°Җ мқјм–ҙлӮ  нҷ•лҘ мқҖ $P(A|B)$мҷҖ к°ҷмқҙ н‘ңкё°н•ңлӢӨ.

л§ҘлқҪмңјлЎң $w_{t-1}$кіј $w_{t+1}$ мқҙ мЈјм–ҙмЎҢмқ„ л•Ң $w_t$к°Җ мқјм–ҙлӮ  нҷ•лҘ мқҖ

$$ P(w_t | w_{t-1}, w_{t+1}) \tag 1 $$

мӢқ 1кіј к°ҷмқҙ м“ё мҲҳ мһҲлӢӨ. мқҙлҠ” CBOWлҘј лӘЁлҚёл§Ғ н•ҳлҠ” мӢқмқҙлӢӨ.

көҗм°Ё м—”нҠёлЎңн”ј мҳӨм°Ё мӢқмқҖ $L = - \sum_k t_k \log y_k$мқҙлӢӨ. $y_k$лҠ” $k$лІҲм§ём—җ н•ҙлӢ№н•ҳлҠ” мӮ¬кұҙмқҙ мқјм–ҙлӮ  нҷ•лҘ мқ„ мқҳлҜён•ҳкі , $t_k$лҠ” м •лӢө л Ҳмқҙлё”лЎң мӣҗн•« лІЎн„°лЎң н‘ңнҳ„лҗңлӢӨ. м—¬кё°м„ң л¬ём ңмқҳ м •лӢөмқҖ $w_i$к°Җ л°ңмғқн•ҳлҠ” кІғмқҙлҜҖлЎң $w_i$м—җ н•ҙлӢ№н•ҳлҠ” мӣҗмҶҢл§Ң 1мқҙкі  лӮҳлЁём§ҖлҠ” 0мқҙ лҗңлӢӨ. мқҙ м җмқ„ нҷңмҡ©н•ҳм—¬, лӢӨмқҢ мӢқмқ„ мң лҸ„н•  мҲҳ мһҲлӢӨ.

$$ L = - \log P(w_t \ | \ w_{t-1}, \ w_{t+1}) \tag 2 $$

мӢқ 2лҠ” мқҢмқҳ лЎңк·ё к°ҖлҠҘлҸ„лқјкі  л¶ҖлҘёлӢӨ. мқҙмІҳлҹј CBOW лӘЁлҚёмқҳ мҶҗмӢӨ н•ЁмҲҳлҠ” мӢқ 1мқҳ нҷ•лҘ м—җ $\log$лҘј м·Ён•ң лӢӨмқҢ л§Ҳмқҙл„ҲмҠӨлҘј л¶ҷмқё кІғмқҙлӢӨ. мқҙлҠ” мғҳн”Ң лҚ°мқҙн„° н•ҳлӮҳм—җ лҢҖн•ң мҶҗмӢӨ н•ЁмҲҳмқҙкі , мқҙлҘј corpus м „мІҙлЎң нҷ•мһҘмӢңнӮӨл©ҙ м•„лһҳ мӢқ 3кіј к°ҷлӢӨ.

$$ L = - \frac{1}{T} \sum_{t=1}^T \log P(w_t \ | \ w_{t-1}, \ w_{t+1}) \tag 3 $$

CBOW лӘЁлҚёмқҳ н•ҷмҠөмқҙ мҲҳн–үн•ҳлҠ” мқјмқҖ мқҙ мҶҗмӢӨ н•ЁмҲҳмқҳ к°’мқ„ к°ҖлҠҘн•ң н•ң мһ‘кІҢ л§Ңл“ңлҠ” кІғмқҙлӢӨ.

skip-gram лӘЁлҚё

skip-gramмқҖ CBOWм—җм„ң лӢӨлЈЁлҠ” л§ҘлқҪкіј нғҖк№ғмқ„ м—ӯм „мӢңнӮЁ лӘЁлҚёмқҙлӢӨ.

CBOW лӘЁлҚёмқҖ л§ҘлқҪмқҙ м—¬лҹ¬ к°ң мһҲкі , к·ё м—¬лҹ¬ л§ҘлқҪмңјлЎңл¶Җн„° нғҖк№ғмқ„ м¶”мёЎн•ңлӢӨ. л°ҳл©ҙм—җ skip-gmm лӘЁлҚёмқҖ мӨ‘м•ҷмқҳ нғҖк№ғмңјлЎңл¶Җн„° мЈјліҖмқҳ л§ҘлқҪмқ„ м¶”мёЎн•ңлӢӨ.

skip-gram лӘЁлҚёмқ„ нҷ•лҘ  н‘ңкё°лЎң лӮҳнғҖлӮҙл©ҙ м•„лһҳ мӢқ 4мҷҖ к°ҷлӢӨ.

$$ P(w_{t-1}, \ w_{t+1} \ | \ w_t) \tag 4 $$

мЎ°кұҙл¶Җ лҸ…лҰҪмқҙлқјкі  к°Җм •н•ҳкі  (л§ҘлқҪмқҳ лӢЁм–ҙ мӮ¬мқҙм—җ кҙҖл Ём„ұмқҙ м—ҶлӢӨкі  к°Җм •н•ҳкі ) мӢқ 4лҘј м•„лһҳ мӢқ 5мҷҖ к°ҷмқҙ л¶„н•ҙн•ңлӢӨ.

$$ P(w_{t-1}, \ w_{t+1} \ | \ w_t) = P(w_{t-1} \ | \ w_{t}) \ P(w_{t+1} \ | \ w_{t}) \tag 5 $$

мң„ мӢқмқ„ көҗм°Ё м—”нҠёлЎңн”ј мҳӨм°Ём—җ м Ғмҡ©н•ҳкі , corpus м „мІҙлЎң нҷ•мһҘмӢңнӮӨл©ҙ м•„лһҳ мӢқ 6мқҙ лҗңлӢӨ.

$$ L = - \frac{1}{T} \sum_{t=1}^T ( \log P(w_{t-1} \ | \ w_{t})+ P(w_{t+1} \ | \ w_{t})) \tag 6 $$

skip-gram лӘЁлҚёмқҖ л§ҘлқҪмқҳ мҲҳл§ҢнҒј м¶”мёЎн•ҳкё° л•Ңл¬ём—җ к·ё мҶҗмӢӨ н•ЁмҲҳлҠ” к°Ғ л§ҘлқҪм—җм„ң кө¬н•ң мҶҗмӢӨмқҳ мҙқн•©мқҙм–ҙм•ј н•ҳлҠ” л°ҳл©ҙ, CBOW лӘЁлҚёмқҖ нғҖк№ғ н•ҳлӮҳмқҳ мҶҗмӢӨмқ„ кө¬н•ңлӢӨ.

лӢЁм–ҙ л¶„мӮ° н‘ңнҳ„мқҳ м •л°ҖлҸ„ л©ҙм—җм„ң skip-gram лӘЁлҚёмқҳ кІ°кіјк°Җ лҚ” мўӢмқҖ кІҪмҡ°к°Җ л§Һкі , corpusк°Җ нҒҙ мҲҳлЎқ м„ұлҠҘ л©ҙм—җм„ң skip-gramмқҙ лӣ°м–ҙлӮң кІҪн–Ҙмқҙ мһҲлӢӨ.

нҶөкі„ кё°л°ҳ vs м¶”лЎ  кё°л°ҳ

нҶөкі„ кё°л°ҳ кё°лІ•кіј м¶”лЎ  кё°л°ҳ кё°лІ•мқё word2vecмқҖ н•ҷмҠөкіј к°ұмӢ  л°©мӢқм—җм„ң м°ЁмқҙлҘј ліҙмқёлӢӨ.

нҶөкі„ кё°л°ҳмқҖ мғҲ лӢЁм–ҙ м¶”к°Җ мӢң мІҳмқҢл¶Җн„° лӢӨмӢң кі„мӮ°н•ҙм•ј н•ҳм§Җл§Ң, word2vecмқҖ кё°мЎҙ к°ҖмӨ‘м№ҳлҘј нҷңмҡ©н•ҙ нҡЁмңЁм ҒмңјлЎң к°ұмӢ н•  мҲҳ мһҲлӢӨ. лӢЁм–ҙмқҳ мң мӮ¬м„ұкіј ліөмһЎн•ң нҢЁн„ҙ мқёмҪ”л”©м—җм„ңлҸ„ word2vecмқҙ лҚ” ліөмһЎн•ң кҙҖкі„лҘј нҢҢм•…н•  мҲҳ мһҲмңјл©°, ‘king - man + woman = queen’ к°ҷмқҖ мң м¶” л¬ём ңлҘј н’Җ мҲҳ мһҲлӢӨ.

к·ёлҹ¬лӮҳ мӢӨм ң мң мӮ¬м„ұ нҸүк°Җм—җм„ңлҠ” л‘җ кё°лІ• к°„ мҡ°м—ҙмқ„ к°ҖлҰ¬кё° м–ҙл өлӢӨ. м¶”лЎ  кё°л°ҳкіј нҶөкі„ кё°л°ҳмқҖ м„ңлЎң кҙҖл Ёлҗҳм–ҙ мһҲмңјл©°, мқҙлҘј л°”нғ•мңјлЎң м¶”лЎ  кё°л°ҳкіј нҶөкі„ кё°л°ҳмқ„ мңөн•©н•ң GloVe кё°лІ•мқҙ л“ұмһҘн•ҳм—¬ л‘җ л°©лІ•мқҳ мһҘм җмқ„ кІ°н•©н–ҲлӢӨ.
